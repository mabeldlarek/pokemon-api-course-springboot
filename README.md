# Pokemon API
The project is simple api for maging Pokemons and their respective reviews. The purpose of development is pratice the use Spring Boot framework.

# Functionalities
- CRUD of Pokemons
- CRUD of Reviews

## Technologies
- Java 17
- Spring Boot 3.1.2
- Gradle
- PostgreSQL / JPA
- Lombok
